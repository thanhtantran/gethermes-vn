Hiển thị tốt trên thiết bị di động
1
Cài đặt Hermes Agent trước
bash 
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
hermes model   # cấu hình nhà cung cấp LLM
2
Clone và khởi động Web UI
bash 
git clone https://github.com/nesquena/hermes-webui.git hermes-webui
cd hermes-webui
./start.sh
hoặc
Cách dùng Docker
bash 
docker pull ghcr.io/nesquena/hermes-webui:latest
docker run -d \
  -e WANTED_UID=$(id -u) -e WANTED_GID=$(id -g) \
  -v ~/.hermes:/home/hermeswebui/.hermes \
  -v ~/workspace:/workspace \
  -p 8787:8787 ghcr.io/nesquena/hermes-webui:latest
→
SSH tunnel cho máy chủ từ xa
bash 
ssh -N -L 8787:127.0.0.1:8787 user@ten-may-chu

Sau đó truy cập http://localhost:8787

Khởi động nhanh

Bắt đầu trong 60 giây

Hoạt động trên Linux, macOS, WSL2 và Android (Termux). Nếu dùng Windows, hãy cài WSL2 trước.

bash — cài đặt một dòng 
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
bash — chạy lần đầu 
source ~/.bashrc      # nạp lại shell
hermes model          # chọn nhà cung cấp LLM
hermes                # bắt đầu trò chuyện
bash — lệnh quan trọng 
hermes model          # chọn nhà cung cấp và model
hermes tools          # cấu hình công cụ bật
hermes gateway setup  # thiết lập nền tảng nhắn tin
hermes doctor         # chẩn đoán sự cố

Tùy chọn hosting

Hermes được thiết kế để tự lưu trữ — chạy trên phần cứng do bạn kiểm soát. Dưới đây là những cách thực tế nhất để triển khai, từ dịch vụ quản lý sẵn đến VPS tự vận hành.

★ Dễ nhất
⚡
HermesBox (sẵn sàng để dùng)
Mua 1 lần dùng mãi mãi

Box Hermes do chính bạn quản lý mà không cần cài đặt, dựa trên mạch nhúng SBC Orange Pi 5 CPU 8 lõi RAM 4GB bộ nhớ 128GB. Đây là nhà cung cấp duy nhất xây dựng riêng cho Hermes, với quyền truy cập terminal đầy đủ, trình duyệt file trực quan và chế độ xem trình duyệt trực tiếp.

  • Chạy riêng trên thiết bị của bạn, đặt tại nhà bạn, hoạt động liên tục bền bỉ
  • Đầy đủ các kênh kết nối WebUI, CLI, Telegram, ...
  • Tích hợp SearXNG instance riêng cho tìm kiếm
  • Dùng API key của riêng bạn (Claude, OpenAI, Gemini, …)
  • Xem trình duyệt trực tiếp: xem tác nhân làm việc theo thời gian thực
Truy cập https://hermesbox.orangepi.vn →
🖥
VPS — tự lưu trữ
~$4–8/mo

Kiểm soát hoàn toàn với chi phí tương đương. Hetzner CX22 là lựa chọn khuyến nghị: 4 GB RAM, 2 vCPU, 40 GB SSD, 20 TB băng thông. Thêm Coolify để deploy một cú nhấp chuột từ GitHub mà không cần làm việc Docker thủ công.

  • Hetzner CX22 (~$4.50/tháng) — hiệu năng/giá tốt
  • Coolify cho deploy GitHub dễ dàng (miễn phí, mã nguồn mở)
  • Vultr, DigitalOcean, Linode đều hoạt động từ $5/tháng
  • Truy cập root đầy đủ, không khóa nhà cung cấp
  • Docker hoặc bare metal, trên hầu hết distro Linux

☁
Nền tảng PaaS
$5–7/mo

Deploy git-push không cần quản trị máy chủ. Tốt nếu bạn muốn bỏ qua hạ tầng hoàn toàn. Railway, Render và Fly.io đều hỗ trợ Node.js và Python với SSL và domain tự động.

  • Railway Hobby — $5/tháng, kết nối GitHub, tự động deploy
  • Render — có gói miễn phí (tạm ngủ khi không hoạt động)
  • Fly.io — tối ưu cho Docker, 30+ khu vực toàn cầu
  • Không cần SSH hoặc quản trị máy chủ
  • CI/CD tích hợp trên mọi git push

Minimum requirements — no GPU needed (Hermes calls external APIs)
  • RAM: 1 GB min, 2 GB recommended
  • CPU: 1 vCPU minimum
  • Storage: 10–20 GB SSD
  • OS: Ubuntu 22.04 LTS or any modern Linux
  • Model: Bring your own key — Claude, OpenAI, Gemini, or local
🏠

Runs anywhere Python 3.11+ runs. Mac Mini, Raspberry Pi, a spare laptop, a homelab NAS — if it can run Python, it can run Hermes. No GPU needed. See the install docs for full requirements.
